# Scientific Calculator (GUI) 🧮

A functional desktop scientific calculator built with **C++** and the **Qt 6 Framework**. This project focuses on event-driven programming and creating a clean, responsive user interface.

## 🚀 Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, Division, and Percentages.
* **Scientific Functions:** Support for Trigonometry (Cos) and Logarithms (log, ln) and power functions.
* **Number-First Logic:** Optimized for immediate computation (e.g., input `90` then press `Cos` for the result).
* **Responsive GUI:** Built using Qt Designer with a Grid Layout for a clean user experience.

## 🛠️ Technical Details
* **Language:** C++
* **Framework:** Qt 6 (Community Edition)
* **Build System:** CMake
* **Signal & Slot:** Leverages Qt’s event-driven architecture to connect UI elements to backend math logic.

## 📁 Project Structure
* `main.cpp`: Entry point of the application.
* `mainwindow.ui`: The GUI design file.
* `mainwindow.cpp`: Implementation of the calculator's arithmetic logic.
* `CMakeLists.txt`: Build configuration file.

## 📝 How to Run
1. Open the project in **Qt Creator**.
2. Configure the project using the **CMakeLists.txt** file.
3. Build and Run the project.
